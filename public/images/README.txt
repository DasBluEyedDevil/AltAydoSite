Ship Images Directory

This directory should contain images for all the ships in the Star Citizen fleet.
Images should be named according to the following convention:

[ship_name_lowercase_with_underscores].png

For example:
- apollo_triage.png
- avenger_titan.png
- constellation_andromeda.png

The image naming pattern follows:
- Convert ship name to lowercase
- Replace spaces with underscores
- Remove apostrophes, periods, and other special characters
- Replace slashes with underscores
- Add .png extension

When a ship image cannot be found, a placeholder image will be shown instead.
